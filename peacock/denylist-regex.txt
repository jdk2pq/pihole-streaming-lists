# Add each line as a separate regex entry in Pi-hole:
#   Admin → Domains → Regex tab → Add to Denylist
#
# NOTE: This pattern is broader than the allowlist pattern and matches the
# ad-serving CDN variants. The allowlist regex (-prd-mc) must take precedence;
# Pi-hole evaluates allowlist regex first, so the setup works correctly when
# both files are applied.

# ── Ad CDN variants (broader than the allowlist pattern) ─────────────────────
# Matches: g<hash>-vod-us-cmaf-prd-<variant>.cdn.peacocktv.com
g[^.]+-vod-us-cmaf-prd-[^.]+\.cdn\.peacocktv\.com
